These are commonly used mathematical HTML entities:

° ± × ÷ ½ ¼ ¾

Usage Examples

  • &deg; - Degree symbol (°)
  • &plusmn; - Plus-minus sign (±)
  • &times; - Multiplication sign (×)
  • &divide; - Division sign (÷)
  • &frac12; - Fraction one half (½)
  • &frac14; - Fraction one quarter (¼)
  • &frac34; - Fraction three quarters (¾)
