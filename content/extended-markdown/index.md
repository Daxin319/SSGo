# Extended Markdown Support

`~~Strikethrough~~` ---> ~~Strikethrough~~

`Sub~script~` ---> Sub~script~

`Super^script^` ---> Super^script^

`==Highlight==` --->  ==Highlight==

```
\`\`\`
Fenced Code Blocks
\`\`\`
```

or

```
\~\~\~
Fenced Code Blocks
\~\~\~ (must match opening delimiter)
```
